HTML

HTML (HyperText Markup Language) is the standard language used to create and structure webpages.

  • HTML uses tags to define elements like headings, paragraphs, links, images, lists, and tables.
  • It forms the skeleton or structure of a website.
  • HTML is not a programming language, but a markup language.
  • It works together with CSS (for styling) and JavaScript (for interactivity).
  • Common tags include "h1-h6" for headings, "p" for paragraphs, "a" for links, "img" for images, and "ul","ol" for lists.
  • Modern HTML5 also includes semantic tags like "header", "footer", "nav", and "section" for better structure and accessibility.

CSS

CSS (Cascading Style Sheets) is a language used to style and design HTML elements on a webpage.

  • CSS controls the appearance: colors, fonts, spacing, borders, backgrounds, and layout.
  • It allows separation of content (HTML) and design (CSS) for cleaner code.
  • Styles can be applied inline, internally inside a <style> block, or externally via a ".css" file.
  • CSS enables responsive designs, making websites look good on different devices.
